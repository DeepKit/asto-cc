## 一、定位

`DM`（差异一元论 / Difference Monism）是整套五层公开主链的**哲学起点层**，没有更上游的理论母层。

最短句：

> **凡能被称为存在者，必已具备可区分性、边界、关系、约束和稳定识别条件。差异不是世界中的第一个对象，而是对象、时间、来源、边界等概念能够成立的前条件。**

DM 的核心台阶：

```
差异（存在的前条件）
  → 边界（结构成其为一）
    → 关系/约束（非任意限制）
      → 稳定继承（痕迹进入后续条件）
        → 实体/存在者（持存的差异结构）
          → 规范/责任（多主体共在的结构后果）
```

每一级不是前一级在时间上的"接下来发生了什么"，而是前一级的结构条件充分到位后，才能开始谈论的下一层描述。

DM 不是：
- 已完成同行评审与长期外部验证的终局理论
- 已封闭所有开放问题的大全体系
- 只要接受就自动让下游一切成立的绝对母法
- 在某个时间开端处有一个"第一个差异"的宇宙生成论

更准确地说：

> **DM 是存在条件论，不是宇宙生成论。它追问的是"什么东西必须已经成立，我们才能开始谈存在者"，而不是"第一个差异如何产生世界"。**

> **DM 是一个以体系化写法呈现、但允许受损、允许修订、允许缩限的研究纲领。**

### 公开定位句

> **差异一元论（DM）是以受非任意约束的可区分性为起点的弱本体论研究纲领。**

### DM-min 与 strong-DM 分层

DM 分部分两层使用：

**DM-min（最小结构接口）**：五个接口字段——`distinguishability` / `non_arbitrary_constraint` / `structure` / `boundary` / `inheritance_trace`。DM-min 只负责提供弱结构语言，不推出意识、主体、价值、规范、责任、政治或文明结论。下游理论只需读取 DM-min 即可运行，不必接受 strong-DM。

**strong-DM（完整研究纲领）**：DM.P00-P11 的完整论证链，涵盖存在论、本体论、时间性、意识、认识、规范、文明韧性等。Strong-DM 是 DM-min 的一个候选解释展开，不是下游理论的必要输入。

> **DM 若受压，首先失去的是 strong-DM 的最强解释权，DM-min 与下游体系不受影响。**

### 三条边界

1. DM 不是已完成同行评审与长期外部验证的终局理论，而是一个可送审、可受损、可继续压缩的研究纲领
2. DM 目前不宣称已经完成意识问题的第三人称闭合，也不宣称已给出制度或文明的终局裁决
3. DM 负责最小本体论与规范地基，不直接替代下游工程、治理、制度设计与场景执行细则

### 写法禁令

以下写法在 DM 全文中统一禁用，需改写为推荐写法：

| 禁用写法 | 推荐写法 |
|----------|----------|
| DM 推出意识 | DM-min 提供结构接口；意识结论需经桥接模型与领域证据 |
| DM 推出责任 | DM-min 提供可区分性与继承痕迹；责任结论需经 TAT 与领域证据 |
| DM 推出政治/文明/工程放行 | DM-min 提供边界语言；具体结论需经 ARBT/ODD 与领域证据 |
| DM 不成立则下游无效 | DM-min 与下游弱连接；strong-DM 受损不拖垮下游 |
| 稳定即正当 | 把"能维持"转入 TAT/伦理制度层，不等于"应当维持" |

### DM-min 最小可保留内核

若 DM 受压，首先缩到以下五个接口：

1. `distinguishability`：至少存在可区分项或可区分关系
2. `non_arbitrary_constraint`：区分有证据、代价或条件支撑
3. `structure`：差异之间可形成可识别组织
4. `boundary`：讨论、责任或适用范围有边界
5. `inheritance_trace`：结构保持、漂移、断裂或升级有可追踪痕迹

> DM 若受压，首先失去的是最强解释权，而不是自动宣告下游体系全部失效。

### 弱依赖定位

DM 是差异一元论的地基层，没有上游理论依赖。

- **独立生存条件**：DM-min 只承诺差异、非任意约束、结构、边界四个最小接口字段，不推出意识、主体、价值、规范、责任、政治或文明结论。
- **下游失效时**：DM 不因下游理论失败而失效。DM-min 可独立作为本体论结构语言使用。
- **reject/defer/escalate**：DM 不输出具体科学结论、临床诊断、法律终裁或工程放行；超出 DM-min 四字段的问题转交下游对应层。

## 二、核心概念

DM 的概念体系围绕以下七个核心概念展开，它们构成从本体到规范的统一语法：

| 概念 | 最低稳定义 | DM 差异点 | C 级 |
|------|-----------|----------|------|
| **差异** | 对象、关系与边界得以成立的最小非同一条件 | DM 把差异当成本体起点，而非语言区分动作或现成实体 | C0 |
| **非任意约束性** | 并非一切组合都同等可行，存在可重复限制关系 | DM 把它当成最低形而上学承诺，而非纯经验归纳结果 | C1 |
| **结构** | 差异项之间形成的非任意约束关联 | 结构有成立条件地位，不只是观察者命名便利 | C0 |
| **边界** | 一个结构成其为一的成员性条件 | 边界是"成其为一"的条件，不只是外部切分线；边界可漂移 | C0 |
| **继承序** | 痕迹继承进入后续成立条件的序位 | 时间性首先表现为继承序，而非先验容器 | C2 |
| **涌现** | 下层结构耦合后形成新的整体边界、整体条件与反向约束 | 新层级必须留下新的约束位置与失效方式，不是超自然跳跃 | C2 |
| **内部性** | 有边界结构的部分状态差异不对称地进入其自身后续成立条件 | 内部性是第一人称问题的最小结构前提，早于自指 | C2 |

### DM 明确拒绝的三条捷径

1. **现成实体起点** — 不把世界的起点设成先验给定的现成物
2. **任意拼装世界** — 不承认一切组合都同等可行
3. **单层平铺解释** — 不把所有层级压成同一平面上的命名便利

### 非任意约束性的起点竞争短表

DM 选择"差异 + 非任意约束性"作为最小起点。为什么不是其他候选？

| 候选起点 | 偷带什么 | DM 的回应 |
|----------|----------|-----------|
| **关系** | 关系预设了关系项的存在，隐含实体承诺 | DM 从差异开始，关系是差异进入约束后的导出 |
| **过程** | 过程预设了时间性，时间性本身需要解释 | DM 先解释继承序，时间性从继承序导出 |
| **事件** | 事件预设了发生框架和区分条件 | 事件是差异结构在继承序中的具体化 |
| **信息** | 信息预设了编码/解码结构和接收者 | 信息是结构关系的度量，不是起点 |
| **约束** | 约束预设了被约束的对象和约束来源 | DM 把约束与差异并列，约束是差异的非任意限制 |
| **属性/状态** | 属性预设了属性承载者 | DM 不承诺先验承载者，结构先于属性归属 |

## 三、推导链

> **以下每一级是结构条件的递进，不是时间序列中"先发生了差异，然后发生了结构"。差异不是时间开端处的第一个事件，而是任何可识别存在者的最低前条件。**

DM 的核心推导按以下逻辑递进，每一环依赖前一环但引入新的约束条件：

```
差异（最小起点）
  ↓ + 非任意约束性
结构（差异进入稳定约束后的组织形态）
  ↓ + 成员性条件
边界（结构成其为一的区分条件）
  ↓ + 位形变换
继承序 / 时间性（痕迹进入后续成立条件的序位）
  ↓ + 持存条件
存在者（在时间中得以持存的差异结构）
  ↓ + 可重复约束关联
规律与因果（结构间可重复的约束关系）
  ↓ + 下层耦合形成新的整体边界与反向约束
层级涌现（新层级出现新的约束位置与失效方式）
  ↓ + 环境变化与扰动
暂稳态 vs 刚性稳定（可转换的暂稳态比刚性稳定更利于长期持存）
  ↓ + 内部性（部分状态差异不对称进入自身后续条件）
生命与主体（内部性在生命层获得具体实现）
  ↓ + 谱系约束、神经-身体耦合
感受、效价、意义与价值（qualia 作为生命主体的短路径暂稳态）
  ↓ + 有限主体对世界的边界化描述
认识与认识边界（认识是受扰动的渐近逼近，非无损镜像）
  ↓ + 多主体共在
规范与责任（对高损伤路径的可归属性标记）
  ↓ + 文明层扩大
制度、纠错能力与文明韧性（危机压缩必须可复归）
```

每一级不是简单的前一级"加上一个新词"，而是引入了新的约束位置和新的失效方式。推导链的强度不在于每一环都逻辑必然，而在于每一环都标出了**失败线**（在什么条件下应缩限或重写）。

## 四、关键命题（P00-P11 主链摘要）

### P00 · 存在论地基

**主张 [C0-C1]**：差异 + 非任意约束性构成 DM 最小起点；时间、持存、规律、因果与涌现是 DM-min 结构接口的导出描述层，不是预先给定的容器或属性。DM 不主张在时间开端处有一个"第一个差异"；差异是可识别存在的最低条件，不是时间上的第一个事件。继承序、时间性、涌现、内部性、责任、文明韧性属于 strong-DM 或 bridge candidate，不属于 DM-min。

**失败线**：若出现更薄、更稳且覆盖同等主链的起点语言，或物理接口长期失配，P00 需缩限或改写。

### P01 · 世界由稳定差异结构构成

**主张 [C2]**：世界应被理解为稳定差异结构的网络，而非先验物体的集合；日常所谓"物"只是对高压缩可持存结构的便捷称呼。

**失败线**：若存在者始终必须重新当作先验实体，结构语言只剩修辞替换，P01 应降格为词汇重写。

### P02 · 差异结构与涌现

**主张 [C2]**：涌现不是神秘跳跃，而是下层耦合后出现新的整体边界、整体条件与反向约束；高层之"新"必须落在可持存的整体条件与对部分状态空间的改写上。

**失败线**：若上层变量总能被一一映射回下层且对部分状态空间没有独立改写，P02 的反向约束主张应降格。

### P03 · 暂稳态与可转换性

**主张 [C2]**：结构跨时间持存不能只靠刚性稳定，还需保留可改道、可缓冲的暂稳态能力；"稳定"不是单一优点，须区分能否承受环境变化与耦合重组。

**失败线**：若刚性结构在跨环境、跨扰动比较中反复更优，P03 的暂稳态优先性应缩限为域内命题。

### P04 · 意识的结构前提与硬问题的重写

**主张 [C2]**：硬问题被重写为——什么样的结构会出现内部性，使得某些状态只能从内部成立位置区分，而不是"物理过程为何神秘伴随体验"。P04 给出的是"可感之位"的结构前提，不是感质的生物闭合。

**失败线**：若纯外部模型能无剩余解释稳定第一人称区分轮廓——即能稳定解释第一人称报告、记忆沉积和后续选择绑定——则 P04 的内部性论证须撤回或重写。

### P05 · 感受、意义、价值与效价

**主张 [C2]**：qualia 收紧为生命主体在谱系约束的神经-身体整体中形成的短路径暂稳态；具体感受、效价、意义与价值必须同时受拓扑、生命史、个体史与文化概念化约束。

**失败线**：若四层半模型关键层长期系统性失配，P05 应缩限甚至总体失败。

### P06 · 感显扰动与认识的边界

**主张 [C2]**：认识不是无损镜像，而是有限主体在感显扰动后对可区分部分作出的边界化描述；知识必然受限不是偶然缺陷而是结构后果。

**失败线**：若边界说明长期无法接到纠错实践与科学共同体结构，P06 需继续收窄。

### P07 · 渐近线推断

**主张 [C2]**：不能占有绝对真理，但可以在持续修正中逼近更稳定的约束结构；P07 解释在认识必然有限的前提下模型为何仍可比较优劣。

**失败线**：若长期无法稳定说明模型优劣区分与残差修正方向，P07 应退回较弱的方法论表述。

### P08 · 责任的拓扑

**主张 [C2]**：责任不是悬空命令，而是多主体共在中对高损伤路径的可归属性标记；规范可以自然化发生，但其效力不能被纯收益语言无剩余吃掉。

**失败线**：若重复互动与归责能力稳定存在却不产生规范沉积，或规范力量被纯收益描述无剩余吃掉，P08 需缩限或重写。

**可比性要求**：P08 的制度与责任判断必须经过跨案例可比性清理——判断所依赖的标准必须在至少两个可区分案例中可复归，不能基于单一案例的独特条件做出普遍性判断。

### P09 · 文明韧性与演化约束

**主张 [C2-C3]**：延续是文明的承载前提（bearer condition），不是最高价值；韧性与纠错是文明续存分析中的核心能力；危机压缩必须可复归。

**失败线**：若高纠错结构在跨案例中稳定构成系统劣势，或永久例外状态长期优于可复归结构，P09 核心制度判断应缩限或重写。

**可比性要求**：P09 的文明韧性判断必须经过跨案例可比性清理——比较标准须显式标注（时间尺度、地理范围、制度类型、可用资源），不能基于不可比案例做出强判断。

### P10 · 与主要传统对话

**主张 [C1]**：DM 与过程哲学、结构主义、非还原物理主义、现象学、实用主义等传统有可对接的近邻概念，但每一处对接都标出了分歧点。

### P11 · 开放边界、主要批评与当代张力

**主张 [C1]**：DM 当前仍有开放项（物理接口、感质闭合、跨作者累积等），体系的强度在于标出自己最可能失败的地方。

## 五、跨层接口

DM 是五层公开主链的起点：

```
DM → ASTO → ARBT → TAT → ODD
```

|----|------|---------------------|
| **DM** | 哲学地基 | — （起点层） |

## 六、失败线总览

DM 不是靠"什么都能解释"来站住，而是靠"知道自己会在哪里失败"来站住。当前最危险的失败路径：

1. 出现比"差异 + 非任意约束性"更薄、却覆盖同等主链的起点语言
2. 物理接口长期悬空，始终无法进入更严格的对接
3. 意识问题的代表作长期不能形成外部可争论入口
4. 规范与制度桥接长期缺少足够硬的案例链
5. 失败条件不足，导致体系越来越像万能解释器
6. 长期停留在"单作者体系 + 内部工具丰富"却没有外部争论与跨作者累积

## 七、术语速查表

以下是 AI 仅凭本文件即可准确使用的核心术语定义。完整术语表见 `DM.P906.术语表.md`。

| 术语 | 定义 |
|------|------|
| 差异 | 对象、关系与边界得以成立的最小非同一条件。DM 的本体起点。 |
| 非任意约束性 | 并非一切组合都同等可行，存在可重复限制关系。DM 最低形而上学承诺。 |
| 结构 | 差异项之间形成的非任意约束关联。 |
| 边界 | 结构成其为一的成员性条件。可漂移，不一定固定。 |
| 位形 | 同一有边界结构在某一方面所取的具体排列或状态。 |
| 继承序 / 时间性 | 痕迹继承进入后续成立条件的序位。DM 先谈继承序，再谈钟表时间。 |
| 偏序 | 成立条件依赖上的严格序位结构，DM 用它压缩时间箭头与继承。 |
| 涌现 | 若干下层结构耦合后，形成新的整体边界、整体条件与对部分的反向约束。 |
| 反向约束 | 整体变量以非一一映射方式改写部分可行状态空间。 |
| 多重可实现性 | 同一上层变量可由多类可区分微观实现族承担。 |

### 起点竞争短表：非任意约束性 vs 其他起点

| 起点候选 | 核心承诺 | 偷带内容（隐含承诺） | DM 的比较优势 | DM 的劣势 |
|----------|----------|---------------------|--------------|-----------|
| **关系**（结构主义/关系本体论） | 实在由关系构成 | 预设关系的可区分性和可重复性 | DM 把可区分性显式化为起点 | 关系论者可能说 DM 是其子集 |
| **过程**（过程哲学） | 实在是过程/事件流 | 预设变化可识别、有方向 | DM 用继承序压缩时间箭头 | 过程论者可能说 DM 太静态 |
| **事件**（事件本体论） | 基本实体是事件 | 预设事件边界可划定 | DM 把边界显式化为问题 | 事件论者可能说 DM 多余 |
| **信息**（信息本体论） | 实在是信息 | 偷带编码者/解码者和信道 | DM 不预设编码主体 | 信息论者有数学工具优势 |
| **约束**（约束本体论） | 约束是基本的 | 偷带被约束者和约束来源 | DM 把约束与可区分性绑定 | 约束论者可能说 DM 多了一步 |
| **差异/可区分性**（DM） | 可区分性 + 非任意约束 | 最少偷带 | 承诺最薄 | 需证明"更薄但覆盖同等" |

> **DM 的核心竞争策略**：起点越薄越好，但必须能覆盖差异→结构→边界→涌现→内部性→规范性全链路。若某个竞争起点能以更薄的承诺覆盖同等链路，DM 应承认并调整。

### 反向约束最小案例模板

判断是否构成反向约束的三条标准：
1. 存在下层组件 A、B，各自独立时可行状态空间为 S_A、S_B
2. A、B 耦合形成整体 G 后，G 具有整体变量 V_G
3. V_G 对 A 或 B 的可行状态空间产生了独立于 A、B 各自约束的新限制

| 案例 | 下层 | 整体变量 | 反向约束表现 |
|------|------|----------|-------------|
| 水分子 | H + O 原子 | 沸点 100°C | H 原子在液态水中的运动自由度受整体相变约束，独立 H 原子无此限制 |
| 交通流 | 各车辆 | 平均速度/密度波 | 单车无拥堵概念；整体密度超过阈值后，每辆车的可行路径被整体变量改写 |
| 组织 | 各成员 | 正式角色/制度 | 个人独立时可自由选择行为；组织整体角色分配后，个人可行行为被角色约束改写 |
| 市场 | 各交易者 | 价格信号 | 单个交易者无法定价格；整体价格形成后，每个交易者的可行交易条件被价格改写 |
| 刚性稳定 | 通过压低变化来维持局部整齐的稳定方式。并非一切稳定的优选形式。 |
| 暂稳态 | 在扰动下倾向回归、但可在大扰动下跃迁的局部稳定区。 |
| 内部性 | 有边界结构的部分状态差异会不对称地进入其自身后续成立条件。第一人称问题的最小结构前提。 |
| 自指 | 结构对自身内部状态的进一步折回处理。晚于内部性，不是起点。 |
| 最小内部性 | 解释稳定第一人称区分轮廓所不可取消的内部可区分状态位置。 |
| qualia / 感质 | 主体内部被经历到的具体感受样式。DM 把它理解为生命主体的短路径暂稳态。 |
| 效价 | 某状态对生命主体来说的正负趋避差异。 |
| 价值 | 生命在多路径后果比较中形成的稳定排序。 |
| 规范性 | 主体间实践结构中成立的"应当"关系位置。 |
| 责任 | 对高损伤结果的可归属性标记。先是可归属性与承接结构，不是直接的道德责备。 |
| 纠错能力 | 共同体保留批评、复审、替代、修正与再分岔的能力。制度和文明韧性的核心变量。 |
| 强否定式 | 为某主张给出可裁决的缩限/重写触发条件。DM 要求自己写出如何被压坏。 |
| 研究纲领 | 以体系化写法呈现、允许开放项和失败条件的可修正主链。 |

### 最易混淆的区分

| 不应混淆 | 区分要点 |
|---------|---------|
| 差异 / 结构 | 差异是起点条件；结构是差异进入非任意约束后的组织形态 |
| 时间 / 过程词 | DM 优先用成立条件依赖与继承序，避免用未压缩的过程词顶替论证 |
| 内部性 / 自指 | 内部性早于自指；自指是内部性进一步折回处理的结果 |
| 暂稳态 / 刚性稳定 | 前者保留跃迁与改道空间，后者优先压低变化 |
| 责任 / 责备 | 责任先是可归属性和承接结构，责备是更高阶的规范判断 |
| 发生学解释 / 正当性结论 | 前者说明"如何长出来"，后者说明"为什么应当被接受" |
| DM-min / strong-DM | DM-min 是最小接口；strong-DM 是完整纲领；下游只需 DM-min |

## 九、最短压缩

> **差异一元论主张：凡能被称为存在者，必已处于差异之中。差异不是世界的第一个事件，而是存在者能够被识别、被描述、被区分的前条件。从差异到结构、边界、继承、存在者、规范，每一级是结构条件的递进，不是时间序列的展开。**

> **DM 是存在条件论，不是宇宙生成论——提供最小结构接口，不替代下游理论的独立论证。**
