## 当前总判断

WSH 当前保持母版封版状态。公开主名为 `和悦论`，应用品牌主句为：

```text
五维归己——给人格自立者的一套五维复盘法。
```

WSH 不判断命运、不定义人格、不替代临床治疗；它把身维、心维、安维、行维与意维中的不舒服，转写为可观察的偏移、可回看的靶心、可执行的调适建议和必要转介。

## WSH-min 治理轨声明

WSH-min 是 WSH 的跨层最小 ABI 接口。治理核心 ABI 冻结为 5 字段，供上下游先判断能否承接：

| field | definition | reject / defer condition |
|---|---|---|
| `WSH.five_dim_vector` | 五维向量（身/心/安/行/意）的偏移方向与幅度 | 缺失任一维度时 defer |
| `WSH.target_core` | 靶心：当前生命阶段的主导性生成课题 | 无法区分“生成的”与“发现的”/“建构的”时 reject |
| `WSH.deviation_snapshot` | 偏移快照：五维偏离靶心的结构描述 | 无靶心参照时 defer |
| `WSH.self_return_degree_G` | 归己度 G：本轮归己回正的状态观察（stable/moving/blocked/refer） | blocked 连续出现或 refer 时不输出独立复盘建议 |
| `WSH.referral_condition` | 转介条件：触发向上转介的红灯条件（近期重大丧失、结构性压迫、临床风险等） | 红灯条件满足时必须转介，不得继续复盘 |

公开复盘输出 ABI 只在未触发转介条件时生成，标准输出对象为 `wsh_reflective_alignment_review`：

| field | definition | reject / defer condition |
|---|---|---|
| `WSH.yearly_target` | 年度靶心 | 靶心被外部权威强行替换时 defer |
| `WSH.monthly_deviation` | 月度偏移：偏离、压抑、失撑、迷茫等偏移观察 | 无靶心参照或用户不同意复盘时 defer |
| `WSH.dual_track_review` | 低阻力发力轨与失衡轨回看 | 轨道判断被写成人格/命运标签时 reject |
| `WSH.boundary_notice` | 边界说明、禁行红线和转介提示 | 边界不明时不得继续承接 |
| `WSH.adjustment_suggestion` | 低风险调适建议 | 触发红灯、临床、法律、责任或高影响请求时 reject |
| `WSH.handoff_target` | `RT6 / COP / TAT / professional_channel / none` | 无法判断承接目标时 defer |

## 跨层消费约束

- **上游（COP/PFM/TAT）**：当 COP 识别到高影响但非责任场景时，通过 `WSH-min` 接口递送，WSH 仅输出复盘假设，不返回责任判定；PFM 的 `pfm_to_wsh_handoff_packet` 只作为候选输入，WSH 必须重新检查边界和用户自愿确认。
- **下游（RT6/LSM）**：WSH-min 输出可被 RT6 的低风险分流和 LSM 的识局假设消费，但 WSH 不承担行动阻塞识别。
- **转介边界**：转介条件触发时，WSH-min 输出 `referral_flag: true`，实际转交流程由 COP 或 PFM 决定。

## 多轨保护口径

WSH 母板不应被 `G 值` 或复盘 Spec 压缩。它至少有三轨：

| 轨道 | 核心资产 | 评价标准 | 保护纪律 |
|---|---|---|---|
| 母板 / 人文理论轨 | 和悦论、五维偏移、归己、靶心、五维传导、人格自立者 | 能命名内在偏移经验、解释生命阶段和自我回归 | 不得因测量学未成熟而删除 |
| 学术 / 测量轨 | G 值、五维四向题项、信度效度、PERMA/Ryff/SDT 对照 | 可测量、可复核、不过度治疗化 | 可以降格权重，但不删除理论生成过程 |
| 应用 / 复盘轨 | 五维归己、低风险复盘、红灯转介、案例与练习 | 能帮助成熟个体自我观察和回看 | 不得替代心理治疗或危机干预 |
